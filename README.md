# Aviva Spa — Demo Website

A luxury spa website built as a client demo. Single-file HTML with no dependencies or build tools required.

## What's included

- Hero section with animated particle background
- Services menu with pricing
- Booking form with date/time selection and validation
- Reviews section
- Contact page with WhatsApp and call buttons
- Mobile responsive

## Stack

- Pure HTML, CSS, JavaScript
- No frameworks, no backend
- Google Fonts (Playfair Display, Cormorant Garamond, Jost)

## Deploy

Works on any static host. Recommended: [Vercel](https://vercel.com) or [Netlify](https://netlify.com) — free tier is sufficient.

## Purpose

Demo site for prospective spa/salon clients. Fully customizable — replace business name, services, pricing, phone number, and colors to match any client.
